# ROS 2 کا تعارف

Robot Operating System (ROS) روبوٹ سافٹ ویئر لکھنے کے لیے ایک لچکدار فریم ورک ہے۔ یہ tools، libraries، اور conventions کا ایک مجموعہ ہے جو مختلف روبوٹک پلیٹ فارمز پر پیچیدہ اور مضبوط روبوٹ رویہ بنانے کے کام کو آسان بناتا ہے۔

ROS 2 اصل ROS کا مکمل redesign ہے، جو جدید روبوٹکس ایپلیکیشنز کی ضروریات کو پورا کرنے کے لیے بنایا گیا ہے، جیسے multi-robot systems، real-time control، اور commercial products۔ یہ Data Distribution Service (DDS) standard کے اوپر بنایا گیا ہے، جو ایک مضبوط اور scalable کمیونیکیشن layer فراہم کرتا ہے۔

یہ باب ROS 2 کا جامع جائزہ فراہم کرے گا، اس کی high-level architecture سے لے کر core communication patterns اور development tools تک۔
