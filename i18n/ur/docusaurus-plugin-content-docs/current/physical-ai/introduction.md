# جسمانی مصنوعی ذہانت کا تعارف

جسمانی مصنوعی ذہانت (فزیکل اے آئی)، جسے اکثر ایمبوڈیڈ اے آئی بھی کہا جاتا ہے، مصنوعی ذہانت کے شعبے میں ایک انقلابی تبدیلی کی نمائندگی کرتی ہے۔ یہ ڈیجیٹل کمپیوٹیشن اور ورچوئل ماحول کی حدود سے باہر نکل کر ایسے ذہین نظام بنانے کے لیے ہے جو مادی دنیا میں ادراک، استدلال اور عمل کر سکیں۔ یہ روبوٹس کا دائرہ کار ہے جو چلتے ہیں، ڈرونز جو اڑتے ہیں، اور خود مختار کاریں جو ہماری سڑکوں پر چلتی ہیں۔
