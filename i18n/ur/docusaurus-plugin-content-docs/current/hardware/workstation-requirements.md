# ورک اسٹیشن کی ضروریات

آپ کا ورک اسٹیشن روبوٹکس ڈیولپمنٹ کا "کمانڈ سینٹر" ہوتا ہے۔ یہی وہ جگہ ہے جہاں آپ کوڈ لکھتے ہیں، سیمولیشنز چلاتے ہیں، اور ڈیٹا کا تجزیہ کرتے ہیں۔ ورک اسٹیشن کی ضروریات آپ کے پروجیکٹس کی پیچیدگی کے مطابق بدل سکتی ہیں، مگر یہاں کچھ بنیادی ہدایات دی گئی ہیں:

- **آپریٹنگ سسٹم**: **لینکس (مثلاً Ubuntu)** روبوٹکس کمیونٹی میں سب سے پسندیدہ آپریٹنگ سسٹم ہے۔ ROS (Robot Operating System) بنیادی طور پر لینکس پر تیار اور ٹیسٹ کیا جاتا ہے، اس لیے بہترین سپورٹ اور مطابقت یہیں ملتی ہے۔
- **پروسیسر (CPU)**: جدید اور ملٹی کور CPU کوڈ کمپائل کرنے اور سیمولیشن چلانے کے لیے ضروری ہے۔ **Intel Core i5/i7، AMD Ryzen 5/7 یا اس کے برابر** ایک بہترین نقطۂ آغاز ہیں۔
- **میموری (RAM)**: روبوٹکس ڈیولپمنٹ خاص طور پر سیمولیشنز کے دوران کافی میموری مانگتی ہے۔ کم از کم **16GB RAM** ضروری ہے، جبکہ **32GB یا اس سے زیادہ** بہتر کارکردگی کے لیے بہت مفید ہے۔
- **گرافکس کارڈ (GPU)**: ایک طاقتور GPU دو اہم وجوہات کے لیے ضروری ہے:
    1. **سیمولیشن**: جدید روبوٹکس سیمولیٹرز جیسے Gazebo اور NVIDIA Isaac Sim رینڈرنگ اور فزکس کیلکولیشنز کے لیے GPU استعمال کرتے ہیں۔
    2. **AI اور مشین لرننگ**: وژن، پرسیپشن اور کنٹرول کے لیے ڈیپ لرننگ ماڈلز چلانا انتہائی کمپیوٹیشنل بوجھ رکھتا ہے، جسے GPU بہترین انداز سے ہینڈل کرتا ہے۔ **NVIDIA کا GPU** CUDA سپورٹ کی وجہ سے سب سے مناسب انتخاب ہے۔ کم از کم **8GB VRAM** والا کارڈ دیکھیں۔
- **اسٹوریج**: ایک **تیز Solid State Drive (SSD)** آپ کے ورک فلو کو نمایاں طور پر بہتر بناتی ہے—بوٹ ٹائم، کمپائل ٹائم اور ڈیٹا لوڈنگ ٹائم سب کم ہو جاتے ہیں۔
