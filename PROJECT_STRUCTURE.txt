API ENDPOINTS:
--------------
Authentication:
  POST   /api/auth/signup          # User registration
  POST   /api/auth/login           # User login
  GET    /api/auth/me              # Get current user

Chat & RAG:
  POST   /api/chatkit/session      # Session placeholder
  POST   /api/query                # RAG chatbot query
  POST   /api/personalize          # Personalize content based on user preferences
  POST   /api/translate            # Translate content to target language
  GET    /api/translate/ur         # Get Urdu intro.md (FIXED)

System:
  GET    /                          # API info
  GET    /health                   # Health check
  GET    /docs                     # FastAPI auto-generated docs
  GET    /openapi.json             # OpenAPI schema

RUNNING THE PROJECT:
--------------------
Backend:
  cd backend
  python -m pip install -r requirements.txt
  # Set up .env file with API keys and database URL
  uvicorn backend.main:app --reload

  OR from project root:
  uvicorn backend.main:app --reload --host 0.0.0.0 --port 8000

Frontend:
  npm install
  npm start  # Starts on http://localhost:3000
