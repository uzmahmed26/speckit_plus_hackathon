BROWSER MEIN YE STEPS FOLLOW KARO:
===================================

STEP 1: Clear Browser Cache (ZAROORI!)
---------------------------------------
Chrome:
  1. Ctrl + Shift + Delete press karo
  2. "Cached images and files" select karo
  3. "Clear data" click karo

Firefox:
  1. Ctrl + Shift + Delete press karo
  2. "Cache" select karo
  3. "Clear Now" click karo

Edge:
  1. Ctrl + Shift + Delete press karo
  2. "Cached images and files" check karo
  3. "Clear now" click karo


STEP 2: Hard Reload (Force Refresh)
------------------------------------
1. Browser mein jao: http://localhost:3000
2. Ctrl + F5 press karo (ya Ctrl + Shift + R)
   (Ye cache ignore karke fresh page load karta hai)


STEP 3: Open Urdu Page
-----------------------
Method A - Language Selector Use Karo:
  1. http://localhost:3000 kholo
  2. Top-right corner mein language dropdown dikhai dega
  3. Dropdown click karo
  4. "اردو" (Urdu) select karo
  5. Page reload hoga Urdu mein

Method B - Direct URL:
  Browser address bar mein type karo:
  http://localhost:3000/ur/docs/intro

  Enter press karo

COMMON ISSUES:
==============

Issue 1: Browser ne old cache store kar rakha hai
Fix: Ctrl + F5 (hard reload)

Issue 2: Service Worker old page cache kar raha
Fix: F12 → Application → Service Workers → Unregister

Issue 3: Wrong URL format
Fix: Ensure URL is: http://localhost:3000/ur/docs/intro
      NOT: http://localhost:3000/docs/ur/intro

Issue 4: Server restart needed
Fix: Terminal mein Ctrl + C, phir npm start phir se
