IF STILL GETTING 404:
=====================

Problem: Specific chapter/page missing Urdu translation
Solution: Check if that specific .md file exists in:
  i18n/ur/docusaurus-plugin-content-docs/current/<chapter-name>/

Example:
If you get 404 on: http://localhost:3000/ur/docs/physical-ai/some-page
Check if this file exists:
  i18n/ur/docusaurus-plugin-content-docs/current/physical-ai/some-page.md

If the file doesn't exist → You need to translate that page
